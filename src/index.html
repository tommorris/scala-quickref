Defining classes

Class names should begin with capital characters as in Java.

class Book {
  def checkOut(borrower: User) {
    // ...
  }
}

Like in Java, classes can be subclasses of other classes:

class Book extends PrintedObject {
  // ...
}

Or of traits:

class Book extends Borrowable {
  // ...
}

A class (or an object) can have use inheritance and traits:

class Book extends Borrowable with Indexable {
  // ...
}

Defining and Using Traits

Traits allow for multiple inheritance while avoiding the diamond problem. Defining a trait is done in a similar fashion to defining a class. Scala allows you to use traits in the same way that one might use interfaces in Java, C# etc.

trait Borrowable {
  def borrow(borrower: User) {
    // ...implementation...
  }
}
